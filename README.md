# ComputerScienceMasters
Exam Notes and Study Notes from Master's Degree in Computer Science @ITU 2025-2027

## 1. Semester
### Exam Study Plan (Subject to change, overlapping)
- `1. December - 16. December` : Algorithm Design
- `17. December - 5. January` : Advanced Programming
- `20. December - 20. January` : Introduction to Machine Learning
- `29. December - 23. January` : Practical Concurrent and Parallel Programming

### Algorithm Design
- **ECTS:** 7.5
- **Exam Format:** Written, on-premises, 4 hours
- **Exam Restrictions:** All physical aids allowed. All digital aids strictly disallowed.
- **Exam Date:** 16. December 2025

### Advanced Programming
- **ECTS:** 7.5
- **Exam Format:** Written, on-premises, 4 hours
- **Exam Restrictions:** All online and physical aids allowed. Use of AI strictly disallowed (i.e. remmeber to diable AI summaries on google, co-pilots in code-editor etc.)
- **Exam Date:** 5. January 2026

### Introduction to Machine Learning
- **ECTS:** 7.5
- **Exam Format:** Oral w/ external examiner
- **Exam Restrictions:** No aids. (But bring Jupyter Notebooks and an itinerary paper)
- **Exam Date:** 20. January 2026

### Practical Concurrent and Parallel Programming
- **ECTS:** 7.5
- **Exam Format:** Oral w/ external examiner
- **Exam Restrictions:** 1 A4 paper, nothing else.
- **Exam Date:** 23. January 2026
